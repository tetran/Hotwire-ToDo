Scene D

Mobile · Drawer open (256px) over 40% backdrop

Trigger / Dismiss

Hamburger (left of header) opens. Three dismiss paths: tap backdrop, ✕ inside drawer, or Esc.

Z-index stack

Backdrop 30 · Drawer 40 · Tooltip 50. Body scroll locked while drawer open.

First-visit default

Mobile drawer is open on first visit so users see the full nav once, then defaults to closed on subsequent loads.

Implementation Spec

Motion

  • · Desktop collapse/expand — 200ms ease-out (width + label opacity)
  • · Mobile drawer slide — 300ms ease-out (transform: translateX)
  • · Backdrop fade — 200ms linear (opacity 0 → 0.4)
  • · Tooltip — instant in / 100ms out
  • · @media (prefers-reduced-motion: reduce) → all transitions become 0ms

Accessibility

  • · Toggle button: aria-expanded, dynamic aria-label
  • · Drawer: role="dialog" aria-modal="true" + focus trap
  • · Esc closes drawer; focus returns to hamburger
  • · Rail icons keep accessible names via tooltip + aria-label
  • · Active item retains visual state (indigo bg + foreground) regardless of collapse

Tokens used

  • · #0f1117 — sidebar bg
  • · #1e2130 — sidebar border / divider
  • · #6366f1 — accent (active fg, logo, focus)
  • · rgba(99,102,241,0.15) — active bg
  • · rgba(255,255,255,0.05) — hover bg
  • · #f8f9fc — content surface
  • · #0f172a — tooltip bg (slate-900)
